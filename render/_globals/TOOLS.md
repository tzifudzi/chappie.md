### Tavily MCP

Tavily MCP provides web search, content extraction, crawling, and site mapping. Use it to gather current information, analyze web content, or explore site structure. Available tools:

- **tavily-search** — AI-powered web search with filters for topic, time range, domain, and depth.
- **tavily-extract** — Extracts and parses content from specific URLs.
- **tavily-crawl** — Crawls a site starting from a base URL, following internal links with configurable depth and breadth.
- **tavily-map** — Maps a site's URL structure without extracting full content.
